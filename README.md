# Travel Blog

This project uses SvelteKit. It is based on the tutorial, [Build a SvelteKit Markdown Blog](https://joyofcode.xyz/sveltekit-markdown-blog), from Joy of Code.

## Setup

Install project dependencies:

```bash
npm i
```

Start a development server:

```bash
npm run dev

# Or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Formatting

Format the entire project source:

```bash
npm run format
```

## Building

Create a production version of the site:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
